------------------------------------------------
OpenCart Module Yandex Sitemap
------------------------------------------------
This simplified site map - XML-file.
For the search engines do not work with the protocol http://www.google.com/schemas/sitemap-image/1.1

Installation
------------
OPENCART VERSION 2:
Unpack archive contents in a folder on your computer.
Download the contents of the folder "upload" in the directory of your Opencart installation.

OPENCART VERSION 3:
You can install the module from the page:
/admin/index.php?route=marketplace/installer
Click the "Download" button and select the "yandexsitemap.ocmod.zip" archive on your local computer.

Then go to the page: 
/admin/index.php?route=user/user_permission/edit
and set permissions for access and modification module for the Administrator group

In the admin panel, go to the page Feeds (admin/index.php?route=extension/extension&type=feed). 
Install the module "Yandex Sitemap" by clicking the "Install" button in front of the module.

Enable module by clicking on the "Edit" button.

The link for the formation of the site map will be displayed in the field "Address:".

Enable module by selecting the mode "enabled" from the dropdown menu and clicking on the "Save" button.
